# LAMP installation
The steps for a simple LAMP stack installation on Ubuntu 18.04

## Specs
* Meant to run with: Ubuntu server 18.04
* Xubuntu-core
* Apache2 + FPM + opcache + configs
* MariaDB
* PHP 7.3 + packages
* Composer & phpcs
* wp-cli + wordpress installer
* Node via NVM
* Python3 & venv
* emacs 26 + Doom
* DBeaver
* Firefox
